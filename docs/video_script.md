## Production Notes

### Visual Elements Needed:
1. Title cards with clean, medical-themed design
2. Screen recordings of the demo interface
3. Agent workflow diagram animation
4. Urgency classification visualization
5. Use case montage footage/images
6. Triage report examples

### Audio:
- Professional narrator (or AI voice like ElevenLabs)
- Subtle background music (medical/tech theme)
- Sound effects for transitions

### Text Overlays:
- Key statistics (e.g., multi-agent workflow, <5 min processing)
- Agent names during workflow
- Urgency levels with color coding
- Call-to-action (GitHub, competition info)

### Pacing:
- Keep energy high but professional
- Use transitions to maintain flow
- Emphasize real-world impact
- Show don't just tell - demo is key

### Accessibility:
- Include captions/subtitles
- High contrast text overlays
- Clear narration
- Visual elements support audio
